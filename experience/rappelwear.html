Overview

RappelWear is a wearable device and app that provides feedback on the user's performance of certain exercises. The shirt is equipped with electrodes that acquire sEMG (surface electromyography) signals and passes them through an analogue circuit. The processed signals are then passed to the app via bluetooth using the Bluno Beetle. Machine learning algorithms then identify the exercise performed and the app provides appropriate feedback to help the user improve their technique!

PCB Design

The printed circuit board RappelWear utilizes was designed in Autodesk's EAGLE software. The design was sent for printing and the printed circuit board can be seen on the bottom left. The components, wires, and electrodes were then soldered onto the board and the final PCB can be seen on the bottom right.

Machine Learning

The ML portion of RappelWear is implemented in Python using the module seglearn which can be found in scikit-learn. The feature data is preprocessed and fed into the pipeline with the appropriate time step and overlap period to produce a discrete probabilistic approximation.
